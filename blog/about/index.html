关于我

本人毕业于浙江万里学院物联网工程专业（软件方向），学历大学本科，2018应届毕业生。
我是从大三下学期开始接触Web前端，当时是通过Web应用与开发这门课，从最开始的 HTML 和 CSS开始入门的，之后就通过慢慢的学习摸索，才入了前端这个坑（哈哈 😆）。
我是一个比较爱折腾的人，喜欢一些能够提升效率、好用、颜值高的工具和软件。对于翻墙这件事也是情有独钟，记得最开始翻墙是看 laod 博客来通过修改 host 来上谷歌的，需要经常去修改，比较烦。之后修改 host 的方法就不能用了，于是就上网搜，主要是使用 VPN 软件和 购买国外服务器来自己搭建，作为一个程序员当然是选择后一种啦 😄 通过网上查找，我选择搬瓦工的服务器，主要比较方便，支持支付宝购买和一间搭建 Shadowsocks 功能，速度不错，从此我就进入到了外面的世界（此处省略800字）。
对于一些工具和软件，我也比较爱折腾，我会选择好几款的软件来进行比较，然后选择一款综合性高的使用。比如 Markdown 软件有很多，收费、免费、离线和在线的，当然我选择的都是免费版本的（学生党比较穷 😢），目前使用比较多的是 马克飞象 和 Typora。开发工具当然选择 vscode 了，选择一款合适的主题和一款舒适的字体还是十分重要的，个人觉得最好看的还是 Material Theme，同时如果哪些样式看着不爽可以到它的配置文件中修改，字体的话我选择的 Source Code Pro 等宽字体。sublime text 和 Atom 我感觉装插件没有 vscode 方便，当然这个还是看个人习惯的，vscode 的集成终端用起来还是很爽的，特别把默认启动改成 bash 就更好了。

还有很多有意思的插件和功能值得去体验和研究！
我的兴趣爱好主要是阅读，爬山，旅游。平时我会看一些自己感兴趣的书籍，技术和非技术的都有。爬山还是意思的，
特别是爬到山顶的那一刻，心情会特别的舒畅，感觉付出的努力会有回报的，我个人比较喜欢爬那种上山下山路线不一样的山。旅游是我非常向往的，游遍祖国的大好江山是我的目标，哈哈。
